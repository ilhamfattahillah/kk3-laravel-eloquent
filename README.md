# Aplikasi Manajemen Karyawan
Project Laravel untuk mengelola data karyawan dan departemen menggunakan Eloquent ORM.

## Fitur
- CRUD Karyawan
- Relasi Karyawan & Departemen
- Search data
- Pagination
